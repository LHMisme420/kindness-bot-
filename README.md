# Kindness Bot ♡

A gentle Python bot that searches for sad tweets on X (Twitter) and replies with random acts of kindness.

## Setup
1. Get API keys from https://developer.x.com (Read + Write permissions).
2. Clone this repo.
3. Set env vars: `API_KEY`, `API_SECRET`, `ACCESS_TOKEN`, `ACCESS_TOKEN_SECRET`, `BEARER_TOKEN`.
4. `pip install -r requirements.txt`
5. `python main.py`

## Messages
Customizable in `main.py` — add your own love!

Spreading warmth, one reply at a time. ❤️
